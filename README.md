## first npm package

### test-package-number-one

## installation

```js
npm i test-package-number-one
```

## Usage

```js
const { addition, multiply } = require("test-package-number-one");
addition(2, 3);
multiply(4, 2);
```

- ES6 Syntax

```js
import { addition, multiply } from "my-test-package-one";

addition(2, 3);
multiply(3, 4);
```
